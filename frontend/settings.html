Local log files

Directory where governance JSON logs (ai_governance.json, escalations.json, …) are written. Relative paths are resolved against the project root.

Splunk HEC integrations

Forward every governance / audit / escalation / error event to one or more Splunk HTTP Event Collectors. Each destination has its own queue and retries, so a slow or down indexer never affects the others or local logging.
